Cross-Walked Geofence DB

A package of data sets containing polygon geometry for political geofences in the United States.

Included data sets cross-walked from top-to-bottom on this package:

  • States
  • Counties
  • Census Tract
  • CSA
  • CBSA
  • Zip Codes
  • Places (localities)
  • Census Block Groups

How does the cross-walk work?

For example, the Census Block Group data includes data from all the other layers, like CSA name and fips code or zip code. Extra data for area type rural or urban and DMA are included.

The polygon and cross-walked data it's included on a GeoJSON file for each layer, for just the cross-walked data a CSV its also included.
